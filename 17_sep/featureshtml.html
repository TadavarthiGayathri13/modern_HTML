Semantics:

Multimedia Support

HTML5 includes native support for audio and video elements,eleminating the need for third-party plugins like Flash.You can use audio and video tags to embed Multimedia media content directly.

Offline web application

HTML5 introduced the Application Cache (AppCache) and Web Storage (localStorage and sessionStorage) mechanisms, which enable web applications to work offline and store data on the client-side. With these features, developers can create web apps that continue to function even when the user is not connected to the internet, improving the user experience and performance
